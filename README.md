# Spice Bowl Payment System

## Overview
This repository contains an e-commerce payment and checkout website developed as part of my coursework while pursuing a Bachelor of Business Administration in Information Systems and Technology (IST).

The website simulates an online checkout process for a spice retailer and demonstrates advanced web form development concepts, including payment processing forms, client-side validation, regular expressions, responsive layouts, and JavaScript form validation.

## Features
* E-commerce payment form
* Billing information collection
* Credit card payment processing interface
* HTML5 form validation
* JavaScript validation testing
* ZIP code validation
* Phone number validation
* Credit card number validation using regular expressions
* Responsive navigation and layout
* Product and recipe content integration

## Technologies Used
* HTML5
* CSS3
* JavaScript
* Regular Expressions
* HTML5 Form Validation
* Responsive Design
